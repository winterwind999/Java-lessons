Abstract Factory Pattern
- Similar to factory
- Factory of factory patterns

Concepts:
- Factory of related objects
- Common interface
- Defer to subclasses

Examples:
- DocumentBuilder
- Frameworks

Design:
- Groups Factories together
- Factory is responsible for lifecycle
- Common interface
- Concrete Classes
- Parameterized create method
- Composition

Pitfalls:
- Complexity
- Runtime switch
- Pattern within pattern
- Problem specific
- Stars as factory then refactored as abstract factory

Pattern Comparison:
- Implemented with a Factory
- Hides Factory
- Abstracts Environment
- Built through Composition
